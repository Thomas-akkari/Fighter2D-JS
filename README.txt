How to use :

- Avoir node installé
- Avoir Internet
- Depuis l'invité de commande, lancer le serveur node :
	Utiliser la commande "node server.js" en étant à la racine du dossier
-Ouvrir localhost:2000 dans votre navigateur (vous pouvez changer le port dans le fichier server.js)

Commandes :
QD ou les flèches pour se déplacer de gauche à droite
Z ou flèche du haut pour sauter
S ou flèche du bas pour frapper 

Fonctions :
Taper "swapPlayers()" dans la console du navigateur pour prendre contrôle de Chun-Li
